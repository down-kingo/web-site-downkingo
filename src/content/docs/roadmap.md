DownKingo is built in public. Here's our development roadmap.
We are constantly working to improve the application based on user feedback. The features listed below represent our current priorities and future vision for the project. By voting on features within the app, you help shape this list.

## ✅ Completed

- [x] **v2 Architecture** — Wails + React foundation
- [x] **SQLite Persistence** — History and queue storage
- [x] **Smart Clipboard Monitor** — Auto-detect copied URLs
- [x] **Internationalization (i18n)** — English and Portuguese
- [x] **Auto-Update System** — Silent background updates

## 🚧 In Progress

- [ ] **Browser Extension** — One-click downloads from your browser

## 🔮 Planned

- [ ] **Plugin Support** — Extend functionality with plugins
- [ ] **Scheduler** — Schedule downloads for later
- [ ] **Cloud Sync** — Sync settings across devices

---

## Build in Public

We believe in transparency. You can:

1. **Vote on Features** — Use the in-app roadmap to vote on what you want next
2. **Track Progress** — See our [GitHub Project Board](https://github.com/orgs/down-kingo/projects/2)
3. **Contribute** — Submit PRs for features you want to see
